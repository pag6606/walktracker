- (decision) Vanilla JS single-file PWA (ADR-01): un archivo, cero build, instalable directo en iOS via Safari
- (decision) localStorage sobre IndexedDB (ADR-02): volumen trivial <100KB/año, API síncrona simple
- (decision) Session como aggregate root (ADR-03): vueltas>=0, sesión finalizada inmutable, distancia siempre derivada, perímetro congelado al cierre
- (decision) Perímetro DERIVADO no almacenado (ADR-04): lapPerimeterM = strideM x stepsPerLap; config guarda medidas crudas
- (decision) Cronómetro wall-clock (ADR-05): elapsedS=(now-startedAt)-totalPausesS; sobrevive freeze de timers iOS; pausa SOLO explícita (no auto-pausa al cambiar de app)
- (decision) Producto ESTRICTAMENTE PERSONAL: no-backend es invariante permanente (no trade-off de v1); sin multi-usuario ni sync cloud
- (decision) Calibración fijada: perímetro = 62 pasos x 0,655 m = 40,61 m (medición real de Paul)
- (decision) Feedback sonoro beep via Web Audio API = Must en v1.0 (háptico descartado: navigator.vibrate no existe en iOS Safari)
- (decision) Despliegue en GitHub Pages (HTTPS estático); SW en sw.js separado; rutas relativas por subpath de proyecto
- (assumption) Schema CSV vs Health Importer diferido a v1.1 (RF-10 ya era Should/v1.1)
- (assumption) Sin métrica cuantitativa de precisión definida (target vs error 8-10% GPS queda como open question)
- (event) PRD finalized — 9 FRs (v1.0), 7 NFRs, 3 OQ, 3 assumptions; pulido editorial aplicado (typo, C7 ref, convencion RF/FR, capitalizacion); cobertura 14/14 RF + 7/7 RNF del SPEC
